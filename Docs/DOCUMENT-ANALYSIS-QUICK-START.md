## Analysis Types

### ?? Generic Options (Always Available)

#### ?? Summarize
- **What**: Creates a concise summary of the document
- **Best for**: Long documents, reports, articles
- **Output**: Paragraph summary

#### ?? Extract Key Points
- **What**: Lists the most important information
- **Best for**: Meeting notes, research papers
- **Output**: Bullet-point list

#### ? Question & Answer
- **What**: Ask specific questions about the content
- **Best for**: Interactive exploration
- **Output**: Specific answers

#### ?? Translate
- **What**: Translate to another language
- **Best for**: Multi-language documents
- **Output**: Translated text

#### ? Custom Analysis
- **What**: Describe your own requirements
- **Best for**: Unique use cases
- **Output**: Custom formatted results

### ?? Specialized Options (Auto-Detected)

#### ? Extract Kursmål (Swedish Course Plans)
- **Triggers when**: Swedish YH or Skolverket course plan detected
- **What**: Extracts Kunskaper, Färdigheter, Kompetenser
- **Output**: Structured list of learning objectives
- **Status**: ? Fully implemented

#### ?? Extract Central Content (Swedish Course Plans)
- **Triggers when**: Course plan detected
- **What**: Extracts "Centralt innehåll" section
- **Output**: Central content items
- **Status**: ?? Coming soon

#### ?? Extract Grade Criteria (Skolverket Plans)
- **Triggers when**: Skolverket plan detected
- **What**: Extracts "Betygskriterier"
- **Output**: Grading criteria by level
- **Status**: ?? Coming soon

### Card States
- **Normal**: White border, transparent background
- **Hover**: Lifts up, blue shadow, blue border
- **Recommended**: Gold border, gold badge, top highlight
- **Custom**: Dashed border

### Navigation Buttons
- **Choose Analysis Type ?**: Proceed from upload to selection
- **Change File**: Go back to upload (keeps nothing)
- **Cancel Analysis**: Stop processing, return to selection
- **Analyze Same File**: Keep file, choose new analysis
- **New Analysis**: Clear everything, start over

## Tips & Tricks

### ?? Tip 1: Start Generic
Upload any document and try "Summarize" first to understand the content.

### ?? Tip 2: Trust the Recommendations
If you see a gold "? Recommended" badge, that's the best match for your document type.

### ?? Tip 3: Multiple Analyses
You can analyze the same file multiple ways without re-uploading.

### ?? Tip 4: Copy & Download
All results can be copied to clipboard or downloaded as .txt files.

### ?? Tip 5: File Size Limit
Max 10MB. If larger, split or extract text first.

## Keyboard Shortcuts

- **Tab**: Navigate between cards
- **Enter**: Select focused card
- **Escape**: Cancel operation (when available)

## Troubleshooting

### "Could not extract text from file"
- **Cause**: Empty file or unsupported format
- **Solution**: Try converting to .txt or .docx

### "No specialized options shown"
- **Cause**: Document type not detected
- **Solution**: Use generic options (they work for any document)

### "Analysis cancelled"
- **Cause**: Exceeded 5-minute timeout or manually cancelled
- **Solution**: Try with a smaller document or simpler analysis

### "Feature coming soon"
- **Cause**: Analysis type not yet implemented
- **Solution**: Check back later or use alternative analysis

## What's Next?

After successful analysis:
1. **Copy**: Paste into your workflow
2. **Download**: Save for later
3. **Analyze Again**: Try different analysis type
4. **New Analysis**: Upload another document
